# Strava brand assets

Official artwork from Strava's brand-assets bundle
(<https://developers.strava.com/guidelines/>), used **unmodified** as required
by Strava's brand guidelines. Do not recolor, alter, or animate these.

- `btn_strava_connect_with_orange.svg` — official **"Connect with Strava"**
  button (orange variant; a white variant is available for dark backgrounds).
  Used for the OAuth connect action.
- `powered_by_strava_orange.svg` — official **"Powered by Strava"** logo
  (horizontal, orange). Shown wherever Strava data is displayed; kept separate
  from and no more prominent than the Freemap logo.

Other guideline points baked into the UI:

- Activities link back with the exact text **"View on Strava"** (`#FC5200`).
- The Strava logo is never used as an app icon or recolored.
